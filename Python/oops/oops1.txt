2Ô∏è‚É£ Class & Object
üîπ Class
=================
A class is a blueprint or template.

It defines properties (variables) and methods (functions).

üîπ Object
==============

An object is an instance of a class.

It represents real data.

üìå Example:

class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def display(self):
        print(self.name, self.age)

# Object creation
s1 = Student("Manohar", 22)
s1.display()

üîç Explanation

Student ‚Üí Class

s1 ‚Üí Object

name, age ‚Üí Properties

====================================================
1Ô∏è‚É£ Instance Variables
=======================
Belong to object

Defined using self

Each object has its own copy

class Employee:
    def __init__(self, name, salary):
        self.name = name        # instance variable
        self.salary = salary    # instance variable

2Ô∏è‚É£ Class Variables (Static Variables)
======================================
Belong to class

Shared by all objects

class Company:
    company_name = "Google"   # class variable

    def show(self):
        print(self.company_name)

3Ô∏è‚É£ Public Properties
======================
Can be accessed anywhere

Default access in Python

class Demo:
    x = 10   # public variable

4Ô∏è‚É£ Protected Properties
===========================
Should be accessed inside class & child class

Prefix with single underscore _

class Demo:
    _y = 20   # protected variable

5Ô∏è‚É£ Private Properties
======================
Cannot be accessed outside class

Prefix with double underscore __

class Demo:
    def __init__(self):
        self.__z = 30   # private variable

    def show(self):
        print(self.__z)
